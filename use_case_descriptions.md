# 팀 코드: `<10281>`

# Use Case Descriptions

## 1. 회원 가입

- **Actor**: 사용자
- **담당**: `<서민지>`
- **Brief description**: 사용자가 ID, 비밀번호, 이름, 전화번호, 이메일을 입력하여 회원으로 가입한다.

| Actor Action | System Response |
|--------------|-----------------|
|              | 1. 회원 가입 화면을 출력한다. |
| 2. ID, 비밀번호, 이름, 전화번호, 이메일을 입력한다. |  |
| 3. 가입 버튼을 누른다. |  |
|              | 4. 가입 완료 메시지를 출력한다. |

**Alternative Courses**
- Step 3. 입력하지 않은 항목이 있는 경우 누락 항목 안내 메시지를 출력한다.

---

## 2. 회원 탈퇴

- **Actor**: 회원
- **담당**: `<서민지>`
- **Brief description**: 회원이 본인 계정을 탈퇴하여 모든 이용 권한과 데이터를 삭제한다.

| Actor Action | System Response |
|--------------|-----------------|
|              | 1. 회원 탈퇴 화면을 출력한다. |
| 2. 탈퇴 버튼을 누른다. |  |
|              | 3. 탈퇴 확인 메시지를 출력한다. |
| 4. 확인 버튼을 누른다. |  |
|              | 5. 탈퇴 완료 메시지를 출력하고 메인 화면으로 이동한다. |

**Alternative Courses**
- Step 4. 취소 버튼을 누르는 경우 탈퇴를 처리하지 않고 탈퇴 확인 메시지를 닫는다.

---

## 3. 로그인

- **Actor**: 회원, 관리자
- **담당**: `<서민지>`
- **Brief description**: 회원 또는 관리자가 ID와 비밀번호를 입력하여 시스템에 로그인한다. 관리자는 ID 'admin', 비밀번호 'admin'을 사용한다.

| Actor Action | System Response |
|--------------|-----------------|
|              | 1. 로그인 화면을 출력한다. |
| 2. ID와 비밀번호를 입력한다. |  |
| 3. 로그인 버튼을 누른다. |  |
|              | 4. 메인 화면으로 이동한다. |

**Alternative Courses**
- Step 3. ID 또는 비밀번호가 일치하지 않는 경우 오류 메시지를 출력한다.

---

## 4. 설문 등록

- **Actor**: 관리자
- **담당**: `<황보민>`
- **Brief description**: 관리자가 설문 제목, 설명, 문항, 응답 항목, 시작/마감 시각을 입력하여 새 설문을 등록한다.

| Actor Action | System Response |
|--------------|-----------------|
|       |  1.설문 등록 화면을 출력한다   |
| 2. 설문 제목, 설명, 문항, 응답 항목, 시작 시각, 마감 시각을 입력한다.        |    |
| 3. 등록 버튼을 누른다.|     |
|       |  4. 등록 완료 메시지를 출력한다.|

**Alternative Courses**
- Step 3. 입력하지 않은 항목이 있는 채로 등록 버튼을 누르면 오류 메시지를 표시한다.
- Step 3. 취소 버튼을 누른 경우 메인 화면으로 돌아간다.

---

## 5. 전체 설문 리스트 조회

- **Actor**: 관리자
- **담당**: `<황보민>`
- **Brief description**: 관리자가 등록된 모든 설문 리스트를 조회한다.

| Actor Action | System Response |
|--------------|-----------------|
|           | 1. 전체 모든 설문 리스트를 표시한다.         |


**Extensions**
- After step 1, 관리자는 상세 내용 보기를 수행할 수 있다. (Use Case 6)
- After step 1, 관리자는 특정 설문 항목 삭제를 수행할 수 있다. (Use Case 7)

**Alternative Courses**
- 등록된 설문이 없으면 "등록된 설문이 없습니다"를 표시한다.
---

## 6. 상세 내용 보기 (`«extend»` of Use Case 5)

- **Actor**: 관리자
- **담당**: `<황보민>`
- **Brief description**: 관리자가 설문 리스트에서 항목을 선택하여 상세 내용을 팝업으로 본다.

| Actor Action | System Response |
|--------------|-----------------|
| 1. 설문 리스트에서 특정 설문 항목을 선택한다.         |           |
|       |   2.  해당 설문의 상세 내용을 팝업 창으로 출력한다.  |
| 3. 닫기 버튼을 누른다           |           |
|       |  4.  팝업을 닫고 설문 리스트 화면으로 돌아간다. |


---

## 7. 특정 설문 항목 삭제 (`«extend»` of Use Case 5)

- **Actor**: 관리자
- **담당**: `<황보민>`
- **Brief description**: 관리자가 설문 리스트에서 특정 항목을 선택하여 삭제한다.

| Actor Action | System Response |
|--------------|-----------------|
| 1.  설문 리스트에서 삭제할 설문의 삭제 버튼을 누른다.         |         |
|        | 2. 삭제 확인 팝업 창을 출력한다.  |
| 3. 확인 버튼을 누른다.  |    |
|        | 4. 해당 설문을 삭제하고 리스트를 갱신하여 출력한다. |

**Alternative Courses**
- Step 3. 취소 버튼을 누른 경우 팝업 창을 닫고 설문 리스트 화면으로 돌아간다.

---

## 8. 설문 검색

- **Actor**: 회원
- **담당**: `<하준수>`
- **Brief description**: 회원이 설문 상태(진행 중/예정/종료)를 선택하고 키워드를 입력하여 설문을 검색하고 결과를 조회한다.

| Actor Action | System Response |
|--------------|-----------------|
|              | 1. 설문 검색 화면을 출력한다. |
| 2. 검색 조건 (현재 진행중, 향후 진행 예정, 종료) 중 하나를 선택한다.| |
| 3. 키워드를 입력하고 검색을 요청한다. | |
|              | 4. 조건과 키워드에 맞는 검색 결과를 화면에 출력한다. |

**Alternative Courses**
- Step 4. 조건에 맞는 설문이 없는 경우 빈 리스트 또는 안내 메시지를 출력한다.

---

## 9. 설문 상세정보 보기

- **Actor**: 회원
- **담당**: `<하준수>`
- **Brief description**: 회원이 검색된 설문 리스트에서 특정 설문을 선택하여 새 화면에서 상세 정보를 조회한다.

| Actor Action | System Response |
|--------------|-----------------|
| 1. 특정 설문을 선택한다. |              |
|           | 2. 설문에 대한 상세 정보(제목, 설명, 문항, 응답 항목, 시작 시각, 마감 시각)를 화면에 출력한다. |
|           | 3. 진행 중인 설문이면 응답 버튼을 활성화한다. |

**Alternative Courses**
- Step 3. 예정 또는 종료된 설문인 경우 응답 버튼을 비활성화한다.

---

## 10. 설문 응답

- **Actor**: 회원
- **담당**: `<하준수>`
- **Brief description**: 회원이 진행 중인 설문에 대해 응답을 작성하고 제출한다.

| Actor Action | System Response |
|--------------|-----------------|
| 1. 응답 버튼을 누른다. |             |
|            | 2. 응답 화면을 출력한다. |
| 3. 설문에 응답한다. |  |
| 4. 응답 버튼을 눌러 완료를 요청한다. |  |
|  | 5. 응답이 정상적으로 제출되었음을 안내하는 메시지를 출력한다. |

**Alternative Courses**
- Step 4. 응답 도중 마감 시각이 지난 경우 마감 안내 메시지를 출력하고 제출을 처리하지 않는다.

---

## 11. 통계 정보 조회

- **Actor**: 관리자
- **담당**: `<서민지>`
- **Brief description**: 관리자가 검색 단위(1주일/1개월/1년)를 선택하여 전체 설문 수, 종료된 설문 수, 총 응답 수 통계를 조회한다.

| Actor Action | System Response |
|--------------|-----------------|
|              | 1. 통계 조회 화면을 출력한다. |
| 2. 검색 단위(최근 1주일, 1개월, 1년) 중 하나를 선택한다. |  |
| 3. 조회 버튼을 누른다. |  |
|              | 4. 해당 기간의 전체 설문 수, 종료된 설문 수, 총 응답 수를 출력한다. |

**Alternative Courses**
- None.
